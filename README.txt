scripting is a Common Lisp ASDF system for generating scripting core
files in the spirit of sbcl-script
(https://github.com/ghollisjr/sbcl-script) and its make-sbcl-core
script.  See sbcl-script's documentation for details on usage and
installation of core files.

Dependencies are quicklisp and the following:

Non-quicklisp dependencies:

* sbcl-script (https://github.com/ghollisjr/sbcl-script)

quicklisp dependencies:

* cl-getopt
* split-sequence
* usocket
* uiop
* cl-fad
* cl-ppcre
* trivial-clipboard

scripting is in the public domain.
